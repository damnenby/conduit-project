# My Plan for Conduit

I want to make this project into a small article app called Conduit.
Right now the project is about books, so I will slowly change it to articles.

The app should let users create an account, log in, write articles, read
articles, leave comments, like articles, and follow other authors. There should
also be a page with articles from followed authors.

I will not try to make everything perfect from the beginning. First I want to
make the basic parts work, and after that I can improve the code and the design.

## Rough steps

- [ ] Look through the existing Books project.
- [ ] Clean up the old Books parts.
- [ ] Start changing the backend to work with articles and users.
- [ ] Add login and registration.
- [ ] Add creating, editing, deleting, and reading articles.
- [ ] Add comments and likes.
- [ ] Add profiles and following authors.
- [ ] Make the Vue frontend pages for these features.
- [ ] Make the app usable on mobile.
- [ ] Add Docker setup.
- [ ] Update the README with how to run the project.
- [ ] Test the main things manually.

I will try to do this in small steps and commit after each finished part.
