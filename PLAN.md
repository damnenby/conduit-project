# My Plan for Conduit

I want to make this project into a small article app called Conduit.
The project started as a books template, but now it is changed to articles.

The app should let users create an account, log in, write articles, read
articles, leave comments, like articles, and follow other authors. There should
also be a page with articles from followed authors.

I will not try to make everything perfect from the beginning. First I want to
make the basic parts work, and after that I can improve the code and the design.
